🔧 ArcShot 滚动截图问题诊断工具

📋 使用说明

  1. 点击"开始诊断"按钮检查当前状态
  2. 查看诊断结果，红色表示需要修复的问题
  3. 按照建议的修复步骤操作
  4. 修复完成后重新测试手动+滚动截图功能

🎯 修复后测试步骤

  1. 打开任意网页（内容要足够长需要滚动）
  2. 点击 ArcShot 扩展图标
  3. 勾选"智能滚动截图"选项
  4. 点击"手动选择区域"按钮
  5. 选择一个区域（确保区域足够大需要滚动）
  6. 等待截图完成，结果页应该显示拼接后的长图

⚠️ 常见问题处理

  • 权限问题：重新加载扩展或重启浏览器
  • 标签页问题：确保原始页面未关闭
  • 超时问题：选择较小的区域或网络较好的页面
  • 数据残留：使用"清除存储数据"按钮
